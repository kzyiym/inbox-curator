# Security Policy

## Reporting a Security or Privacy Issue

If you discover a security vulnerability or a privacy concern in Inbox Curator, please report it responsibly.

**For sensitive issues**, please use [GitHub Security Advisories](https://github.com/kzyiym/inbox-curator/security/advisories/new) if enabled. Otherwise, open a minimal public issue **without** including any sensitive data.

**When reporting, include:**

- Plugin version
- Obsidian version
- Operating system (Windows, macOS, Linux)
- AI provider (OpenAI / Gemini / Anthropic)
- Review mode (Advanced / Auto-sort / Review only)
- Whether auto-sort is enabled
- Whether image review, PDF extraction, or URL extraction are enabled
- Steps to reproduce
- Expected behavior vs. actual behavior
- Logs or screenshots (redacted)

**Do NOT include in any report:**

- API keys
- Note contents
- Vault file paths
- Local file system paths
- Private URLs
- Provider account details or credentials

Thank you for helping keep Inbox Curator secure.
